# Contributing | 如何貢獻

感謝你對本清單的關注！

## 如何提交你的伺服器

本清單的伺服器資料由 [HiddenHosts](https://hiddenhosts.com) 自動同步。

要將你的伺服器加入清單：

1. 前往 [HiddenHosts](https://hiddenhosts.com) 註冊帳號
2. 在控制台中提交你的伺服器資料
3. 伺服器通過審核後會自動出現在本清單中

## 回報問題

如果你發現清單中有任何錯誤或過時的資訊，請：

- 在本 repo 開一個 [Issue](../../issues)
- 或到 [HiddenHosts](https://hiddenhosts.com) 上回報
